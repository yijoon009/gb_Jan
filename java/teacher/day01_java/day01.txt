클래스(반)

	타입이다.
	공통요소를 매번 새로 선언하지 않고
	하나의 클래스 안에서 한 번 선언하여
	가져다 사용한다.

객체(인스턴스 변수)
	추상화된 개념을 구체화 시키는 것.
	클래스의 필드에 접근할 수 있는 변수.

객체화(인스턴스)
	객체를 만드는 작업.
	추상적인 개념을 구체화 시키는 작업.

	Car   mom = new   Car();
	클래스 객체       Heap  할당된 필드의 주소

※ . : 하위 연산자, 멤버변수 접근 연산자
	객체명.필드명
	객체명.메소드명()

생성자
	클래스명()

	이름 뒤에 소괄호가 있기 때문에 메소드와 동일한 기능을 가지고 있다.
	하지만 생성자는 리턴이라는 기능을 제공하지 않기 때문에 메소드라고 부르면 안된다.

생성자의 목적
	1. 해당 클래스 필드를 메모리에 할당한 후 할당된 주소값을 리턴한다.
